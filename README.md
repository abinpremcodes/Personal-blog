# Personal Blog

A blogging platform built with Django where users can create, edit, and delete blog posts.

## Features

- User authentication (register, login, logout)
- Create blog posts with images
- Edit and delete own posts
- Categories and search functionality
- Comment system
- Responsive design with Bootstrap

## Technologies

- Python
- Django
- HTML/CSS
- Bootstrap
- SQLite
